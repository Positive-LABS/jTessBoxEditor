jTessBoxEditor
==============

Optimize [jTessBoxEditor](http://sourceforge.net/projects/vietocr/files/jTessBoxEditor/) for Tesseract.

## Added

+ ShortCuts
  - Alt+S Prev Box
  - Alt+D Next Box
  - Alt+F Modify Character
  - Alt+H  X--
  - Alt+J  X++
  - Alt+I  Y--
  - Alt+K  Y++
  - Alt+N  W--
  - Alt+M  W++
  - Alt+L  H--
  - Alt+,  H++

+ Delete boxes in a line
